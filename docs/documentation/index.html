What is Liberatube?
Liberatube is a privacy, feature rich alternative front end to YouTube that uses the PHP scripting languange as the backend.

Installation

To install Liberatube first you need to get a program that can run PHP scripts or use a Hosting provider that can PHP scripts, we recommend using apache2 with a combination with PHP, and for external hosts we recommend using InfinityFree because it's free.
You also need a SQL capable database program, we recommend using MySQL, most Hosting providers that have PHP already support MySQL.
When you have done that you want to download your desired version of Liberatube from the GitHub and extract the contents to the root of the website docs.
After you have done that you need to edit config.php and configure the database details, you need to configure what user has access to the admin dashboard, if you want, you can configure the different Invidious instances.
